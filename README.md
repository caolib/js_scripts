# 一些自用的油猴脚本

多数功能都可以单独开启和关闭，在插件面板点击对应功能选项即可关闭
![image-20260226133708478](https://free.picui.cn/free/2026/02/26/699fdbb96ce02.png)

## 1.github

### GitHub Release 增强显示

1. 文件下载量显示：添加获取下载量按钮，点击可以获取并显示所有文件的下载量
2. 安装包显示优化：对文件进行分组显示，根据用户系统类型将对应平台的安装包置顶
3. 代理按钮合并：将脚本 [Github 增强 - 高速下载](https://greasyfork.org/zh-CN/scripts/412245-github-enhancement-high-speed-download) 的下载按钮合并显示为下拉列表

![image-20260226014141704](https://free.picui.cn/free/2026/02/26/699f340a23384.png)
