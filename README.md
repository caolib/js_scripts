# 我自用的一些油猴脚本

## github

### GitHub Release 增强显示

1. 文件下载量显示：添加获取下载量按钮，点击可以获取并显示所有文件的下载量
2. 安装包显示优化：对文件进行分组显示，根据用户系统类型将对应平台的安装包置顶
3. 代理按钮合并：将脚本 [Github 增强 - 高速下载](https://greasyfork.org/zh-CN/scripts/412245-github-enhancement-high-speed-download) 的下载按钮合并显示为下拉列表
